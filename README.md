# zhihu-daily-simple

> A Vue.js project created with vue-router and vuex to read articles of Zhihu Daily in a simpler way

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
